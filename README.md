Project Overview

This is a simple number guessing game where the player tries to guess a randomly generated number within a set range (e.g., 1 to 100).
The program provides feedback to the player by telling them if their guess is too high or too low. 

How to Play
Run the program.
The game will generate a random number.
Enter your guess when prompted.
The game will tell you if your guess is "too high" or "too low".
Continue guessing until you find the correct number.
The game congratulates you and shows the number of attempts it took.
